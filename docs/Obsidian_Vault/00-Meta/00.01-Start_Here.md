# Start Here - Índice Principal del Proyecto

Bienvenido al **Vault del Planificador Inteligente** del Molino Fábrica Cultural. Este espacio está diseñado para brindarle contexto inmediato a cualquier Agente o desarrollador sobre el funcionamiento interno de la aplicación.

## 🧭 ¿De qué se trata este proyecto?
Es una aplicación web PWA (Progressive Web App) diseñada para gestionar la asignación de roles, dispositivos, capacitaciones y control de asistencia de los residentes (agentes) en los diferentes espacios (dispositivos) del Molino Fábrica Cultural.

## 🗺️ Rutas de Lectura Sugeridas

Si eres un **nuevo agente** asumiendo el control de esta sesión, dependiendo de la tarea que debas hacer, sigue estas rutas de lectura para ahorrar tokens y cargar solo el contexto necesario en tu memoria:

### 1️⃣ Onboarding Global (Entendiendo todo)
Si necesitas entender el ecosistema y cómo se interconectan el frontend y la base de datos:
1. Lee este archivo (Estás aquí).
2. Ve a [[01.01-Vision_General]] para entender el Stack Técnico.
3. Ve a [[02.01-Esquema_Tablas]] para entender el modelo de datos.
4. Ve a [[03.01-Hooks_Core]] para ver cómo React hace fetch de todo.

### 2️⃣ Bugs de Frontend / Modificación de UI
Si el usuario pide un cambio en las vistas, colores, o comportamiento de los paneles:
1. Revisa [[03.02-Componentes_Principales]] para identificar el archivo correcto (`PlanningMatrix`, `ExecutionTab`, etc).
2. Lee [[04.01-Reglas_Asignacion]] para asegurar que tus cambios visuales no rompan las reglas de negocio (ej. límites de cupo).

### 3️⃣ Integración de Datos, Supabase o Problemas de Guardado
Si hay problemas donde "se guardan mal los datos" o necesitas crear una nueva tabla:
1. Lee [[02.01-Esquema_Tablas]] rigurosamente.
2. Continúa con [[02.02-Funciones_RPC]] si la lógica es pesada.
3. Chequea [[05.01-Bugs_Conocidos]] (especialmente los referentes a permisos y caché de Supabase PostgREST).

---
**Nota para Agentes:** Siempre busca reutilizar las convenciones actuales antes de introducir dependencias nuevas. Revisa la carpeta `00-Meta` por instrucciones de código.
